Phaser 遊戲程式教學 - 火箭遊戲

在這個教學中，我們將學習使用 Phaser 遊戲引擎來製作一個簡單的火箭遊戲。遊戲的目標是讓火箭躲避炸彈並獲得分數。

遊戲元素介紹

  • 火箭：玩家操控的角色，通過滑鼠控制火箭的移動。
  • 炸彈：隨機從上方落下的障礙物，玩家需要躲避炸彈。
  • 分數：當火箭成功躲避炸彈時，玩家獲得一分。

程式碼解析

在這個遊戲中，我們使用了幾個重要的程式碼片段來控制遊戲的邏輯：

// 創建火箭物件 this.rocket = new Rocket(this); // 創建炸彈物件 let newBomb = new Bomb(this); // 滑鼠事件檢測 this.input.on('pointermove', function (pointer) { // 控制火箭移動 self.rocket.setVelocityX((pointer.x - self.rocket.x) / 2); });

遊戲流程

遊戲開始時，炸彈會隨機從上方落下，玩家需要移動火箭來躲避炸彈。當火箭成功躲避炸彈時，玩家獲得一分。如果炸彈撞到火箭，遊戲結束。
