Phaser 遊戲程式教學

如果你想學習如何使用 Phaser 框架來開發遊戲，本教學將引導你進入 Phaser 的世界，做一個簡單的遊戲。

重要的概念

在 Phaser 中，有幾個重要的概念需要理解：

  • Scene（場景）：代表遊戲中的一個場景，包含遊戲中的元素和動作。
  • Sprite（精靈）：在遊戲中可以移動的圖像，如角色、子彈等。
  • Group（羣組）：用來包裝多個精靈，方便管理與處理碰撞事件。
  • Physics（物理引擎）：處理對象之間的碰撞和運動。
  • Animation（動畫）：讓精靈有生動的表現，如移動、攻擊等。

程式碼範例

以下是一個簡單的遊戲程式碼範例：

        
            // 在 create() 方法中建立場景
            create() {
                // 加載遊戲資源
                this.load.image("rocket", "rocket.png");

                // 創建精靈
                this.rocket = this.physics.add.sprite(400, 300, "rocket");

                // 設定物理效果
                this.rocket.setCollideWorldBounds(true);

                // 創建羣組
                this.bullets = this.physics.add.group();
                this.enemies = this.physics.add.group();

                // 設定碰撞事件處理
                this.physics.add.overlap(this.bullets, this.enemies, this.bulletHitEnemy, null, this);
            }

            // 在 update() 方法中更新遊戲狀態
            update() {
                // 檢查按鍵輸入
                if (this.input.keyboard.checkDown(this.cursors.space, 500)) {
                    this.fireBullet();
                }
            }
