Phaser 遊戲程式教學

歡迎來到 Phaser 遊戲程式教學網頁！在這裡，我將會逐步介紹如何使用 Phaser 創建一個遊戲場景，包括預加載素材、創建角色、設定物理碰撞等。讓我們開始吧！

預加載素材

在遊戲開始前，我們需要預先加載遊戲所需的素材，例如圖片、動畫等。以下是一個 preload 函式的範例，用來加載火箭圖片：

    <script>
      preload() {
        this.load.image('rocket', 'assets/rocket.png');
      }
    </script>

創建遊戲場景

接下來，我們需要創建一個遊戲場景（Scene），並在其中添加遊戲物件，例如火箭、敵人等。以下是一個 create 函式的範例，用來創建火箭物件：

    <script>
      create() {
        this.rocket = new Rocket(this, 100, 300);
        this.physics.add.existing(this.rocket);
      }
    </script>

物理碰撞處理

在遊戲中，我們需要處理物體之間的碰撞事件，例如火箭與敵人的碰撞。以下是一個物理碰撞處理的範例，用來處理火箭與敵人碰撞時的事件：

    <script>
      update() {
        this.physics.add.overlap(this.rocket, this.enemies, function(rocket, enemy) {
          // 在碰撞時觸發的事件
        });
      }
    </script>
