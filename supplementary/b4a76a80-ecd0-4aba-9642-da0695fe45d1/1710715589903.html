Phaser 遊戲程式教學 - Cursor 與 Animation

在 Phaser 遊戲開發中，使用 Cursor 控制角色移動是非常常見的操作，同時在遊戲中加入動畫效果也能提升遊戲的趣味性。

Cursor 控制角色移動

在 Phaser 中，可以使用 Cursor 鍵來控制角色的移動。首先，我們要在 preload 函數中載入我們的角色圖片：

this.load.spritesheet("dude", "path/to/dude.png", { frameWidth: width, frameHeight: height });

接著，在 create 函數中放置我們的角色：

this.player = this.physics.add.sprite(x, y, "dude");

最後，在 update 函數中設置 Cursor 鍵的事件，來移動角色：

update() {
    if (this.cursors.left.isDown) {
        this.player.setVelocityX(-speed);
    } else if (this.cursors.right.isDown) {
        this.player.setVelocityX(speed);
    } else {
        this.player.setVelocityX(0);
    }
}

加入角色動畫

要為角色添加動畫效果，首先我們需要在 preload 函數中載入角色的 spritesheet

this.load.spritesheet("dude", "path/to/dude.png", { frameWidth: width, frameHeight: height });

然後在 create 函數中創建角色的動畫：

this.anims.create({
    key: "left",
    frames: this.anims.generateFrameNumbers("dude", { start: 0, end: 3 }),
    frameRate: 10,
    repeat: -1
});

最後，在 update 函數中根據角色的移動來播放對應的動畫：

update() {
    if (this.player.body.velocity.x < 0) {
        this.player.anims.play("left", true);
    } else {
        this.player.anims.play("right", true);
    }
}
